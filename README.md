### What I'm building

**[DeployLog](https://deploylog.dev)**: release notes from your deploys. Full-stack SaaS built end-to-end: a hosted page, an embeddable widget, an npm CLI that turns git commits into release notes with an LLM, and a GitHub Action. Launching.

**Habitagram**: habit tracking as a journey. A React Native app where your habits move a boat across illustrated landscapes that shift with the time of day. Near launch.

**Studio**: the generative media engine behind both. Procedural numpy/scipy/PIL engines that render their art, video, sound, and music from code, no stock assets.
